## Projects

**AI · performance · local-first**
- 🎙️ **[murmur](https://github.com/JakubGawr/murmur)** — A local-first macOS app that records meetings, transcribes them **on-device** (whisper.cpp on Metal), and turns each one into a clean note that lives in your Obsidian vault. An on-device brain — embeddings, semantic search / RAG over your own notes, NER, a pluggable LLM provider — does the thinking locally; nothing leaves the machine unless you say so. Built on **Tauri 2 (Rust)** + **Angular 18 zoneless**, over a SQLCipher-encrypted store with a per-folder Touch ID lock.

## What I care about

- ⚡ **Performance** — latency budgets, zero-copy paths, doing the work on-device instead of round-tripping a server.
- 🤖 **AI on the edge** — local models + on-device embeddings, semantic search / RAG over your own data, redaction firewalls, agentic workflows that respect it.
- 🏛️ **Architecture** — SQL as the single source of truth, thin readers, seams you can swap a provider through without ripping out the core.
